In TypeScript, the ! (exclamation mark) after a variable name is called the definite assignment assertion operator. This operator tells TypeScript that you are sure this variable will be assigned a value before it is accessed, even though TypeScript's normal flow analysis might not be able to determine this.

id! is the variable name with the definite assignment assertion operator.
: number specifies that id is of type number.
